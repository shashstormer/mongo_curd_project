What is CRUD?

CRUD is an acronym for Create, Read, Update, and Delete. CRUD operations are basic data manipulation for a database.

How to use the CRUD feature?  *This assumes you have set up the server and it is running

GET :
When you fetch the /crud route with GET method the server retrives data from the database and returns it. Refer here for payload information.

PUT :
When you fetch the /crud route with PUT method the server inserts data into the database and returns the item ID. Refer here for payload information.

PATCH :
When you fetch the /crud route with PATCH method the server updates data in the database and returns the old data. Refer here for payload information.

DELETE :
When you fetch the /crud route with DELETE method the server deletes data from the database and returns the deleted data. Refer here for payload information.
